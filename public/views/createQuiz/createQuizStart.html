How To Create Quiz:

  1. Create results: theses are the final characters, traits, or attributes the quiz taker can get (Ned Stark, INTJ...)
  2. Create Questions: write each question that is included in the quiz
  3. For each question add answers: these answers will have one (or multiple) results that they give towards and a weight that they add to that result's final score.
  4. Publish quiz: Once you're done creating a new quiz, publish it so others can take it!
